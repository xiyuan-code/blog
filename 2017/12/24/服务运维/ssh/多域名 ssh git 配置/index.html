ssh config 配置. ~/.ssh/config

  • Host >域名
  • User >用户名
  • IdentityFile >ssh key的路径
  • IdentitiesOnly >只使用这里设置的key, 防止使用默认的
  • ServerAliveInterval >连接保持
  • ControlMaster auto >不用重新登录
  • ControlPath ~/.ssh/master-%r@%h:%p
